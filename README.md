# quiple.dev

## 요구사항
* node.js
* pnpm

## 빌드하기
```sh
pnpm i
pnpm build
```

## 개발용 서버 실행
```sh
pnpm i
pnpm build
pnpm preview
```

`pnpm prevoew` 실행 후 `/chzzk`에 치지직으로 로그인하여 쿠키를 생성했다면 이후에는 `pnpm dev`로 개발 가능.
